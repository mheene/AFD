The host.info file

There are two possibilities to hold information of a certain host. The first is in <host alias name> in directory $AFD_WORK_DIR/etc/info. Everything stored in this file will be shown in afd_info dialog. Another method is to store these in the central $AFD_WORK_DIR/etc/host.info file. The format of this file is as follows: 

          <alias name 1>
          Text to show.
          </alias name 1>
          .
          .
          .
          <alias name n>
          Text to show.
          </alias name n>

Its also possible to specify both types, it will then first look into $AFD_WORK_DIR/etc/host.info and if the host alias name is not found in there try to get the information from $AFD_WORK_DIR/etc/info/<host alias name>.
